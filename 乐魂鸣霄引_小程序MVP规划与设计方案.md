## 一、 项目背景与产品定位

### 1.1 项目背景
《乐魂鸣霄引》是一个融合盛唐文化与传统乐器拟人化的国漫与CV（声优）创新项目。针对国漫市场同质化严重、传统乐器文化传承受限等痛点，项目以古琴、琵琶、阮、二胡、埙、竖笛、筑、篪等八种乐器化灵为主角，结合天宝年间历史背景，打造“声画同构”的跨媒介叙事生态。

### 1.2 产品定位
本小程序定位为**轻量化非遗数字化展示与体验平台**。依托现有的核心IP资产、视频内容与原创剧本，打造集IP展示、内容传播、剧本沉浸与轻互动体验于一体的综合载体，作为参加创新创业大赛的成果展示窗口，并承接线上线下流量。

### 1.3 目标用户
- **国风动漫爱好者**（15-35岁）：了解原创IP形象、观看动画样片、阅读剧情。
- **赛事评委与投资人**：快速了解项目全貌、评估IP资产价值与商业潜力。
- **高校与社区合作方**：了解合作模式、预约非遗课程或工作坊。

## 二、 核心 IP 形象落地分析

项目已完成八位核心角色（七位主角团+一位反派）的详尽人设与视觉设计，具备极高的落地可行性：
1. **文化底蕴深厚**：角色设定扎根盛唐历史，如谢云止融合道教音律与宫廷雅乐，薛枕月承载将门风骨与教坊司沧桑。
2. **视觉辨识度高**：传统服饰、纹样（饕餮纹、忍冬纹等）与乐器特征巧妙结合，符合“国潮”审美，便于实体周边开发。
3. **跨媒介延展强**：“CV声画同构”赋予角色独特声音灵魂，为动画、广播剧及虚拟偶像运营打开想象空间。

## 三、 小程序功能模块规划 (MVP 版本)

遵循“重展示、轻交互、易传播”的设计原则，MVP 版本聚焦三大核心板块。

### 3.1 首页：总览与导流枢纽
- **顶部 Banner**：轮播展示核心IP海报、动画封面、剧本宣传图，配以核心文案。
- **核心入口**：四个图标按钮直达【乐魂IP馆】【声影剧场】【鸣霄剧本阁】【互动体验坊】。
- **项目简介**：简炼介绍“原创IP+盛唐文化+CV演绎”核心卖点。
- **角色速览**：横向滑动展示首发三位角色的Q版头像与简介，点击进入详情。

### 3.2 核心板块一：乐魂IP馆 (核心资产展示)
- **IP 全景画卷**：长卷轴横向展示角色群像立绘，阐述“乐器化灵”初衷。
- **角色详情页**：
  - 高清立绘与乐器设计图
  - 背景故事与性格解析
  - 武器/能力介绍
  - 服饰纹样解读（凸显文化考究）
  - 参考古诗与细节彩蛋

### 3.3 核心板块二：声影剧场 (视听内容承载)
- **动画样片展映**：集中展示已制作的动画概念片、角色动态展示。
- **幕后创作纪实**：画师绘画过程、CV配音花絮、团队工作日常。
- **轻量互动**：视频点赞、收藏，简单的文字评论区。

### 3.4 核心板块三：鸣霄剧本阁 (原创剧情阅读)
- **世界观导读**：阐述天宝年间背景、“乐器化灵”机制与“雅俗之争”矛盾。
- **剧情高光节选**：图文结合展示最具张力的片段（如教坊司连环命案开端），关键处设置悬念。
- **创作手记**：编剧分享如何将乐理融入悬疑剧情，提升文化公信力。

### 3.5 辅助板块：互动与资源对接
- **互动体验坊**：提供角色线稿下载用于涂色创作，发布共创征集。
- **合作预约**：极简表单收集合作意向（校园课程/社区活动/商务合作）。
- **关于我们**：团队简介与版权声明。

## 四、 视觉设计规范

- **色彩体系**：
  - 主色调：朱红 (`#C41A1A`)
  - 辅助色：墨黑 (`#1A1A1A`)
  - 点缀色：鎏金 (`#C9A96E`)
  - 背景色：宣纸白 (`#F5F0E8`)
- **字体规范**：标题使用国风韵味衬线体（如思源宋体），正文使用无衬线体（如思源黑体）。
- **装饰元素**：边角、分割线运用盛唐经典纹样（忍冬纹、宝相花纹），卡片边框参考皮影镂空风格。

## 五、 MVP 开发实施建议

为快速落地并控制成本，MVP 版本建议采用 **Web-Static 静态单页应用** 架构进行模拟展示。

1. **技术栈**：Vite + React + TypeScript + TailwindCSS。
2. **核心目标**：实现首页导流、IP馆图文展示、剧场视频模拟播放、剧本阁图文阅读的核心链路。
3. **内容策略**：首批仅填充谢云止、薛枕月、麦勒三位角色的详细数据与少量视频/剧本占位内容，确保快速跑通流程。
